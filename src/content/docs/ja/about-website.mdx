本网站主要用来收集张祥前的《统一场论》等诸多著作。大家可以无偿免费观看。我会在其中补充制作一些动态的演示动画。

所有我添加的部分，都会有类似下面的 `说明` / `标记`。除此之外，其余内容都来自张祥前的著作。

:::note
有这样的 `说明` / `标记` 表示这部分内容是 **[uft@obiscr.com](mailto:uft@obiscr.com)** 添加，并非来自张祥前。
:::

值得一提的是，我添加的这部分内容，可能有因为理解不到位而制作错误的地方。如果发现了错误，请[在这里报告错误](https://github.com/unified-field-theory-org/documentation/issues/new?title=%E5%8F%91%E7%8E%B0%E5%8A%A8%E7%94%BB%E6%B8%B2%E6%9F%93%E9%94%99%E8%AF%AF&labels=animations)，非常感谢！

## 什么时候更新

一般会每隔 2～3 天更新一次。闲暇之余制作，无法保证稳定的更新频率。
